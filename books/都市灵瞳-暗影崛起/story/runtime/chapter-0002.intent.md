# Chapter Intent

## Goal
让灵瞳在地铁站落地，验证功能并埋下代价

## Outline Node
五卷，每卷约 100 章。第一卷"压抑与试探"，情绪曲线是低开高走——开篇即生死危机，林玄在公司加班时胸口剧痛，灵瞳首次被动开启，看到地铁站方向有异常灵脉波动，三年倒计时从这一刻开始具象化；中间压抑蛰伏，主角在灵管局监视下小心翼翼地寻找第一个灵脉节点，每一次灵瞳使用都伴随着生理痛苦；卷尾第一次小爆发，林玄在公共场合被迫使用灵瞳，引起灵管局注意，身份从"隐藏者"变为"被监视者"。第二卷"暗流与同盟"，情绪曲线是波动上升——林玄与苏晚晴结成利益同盟，但每一次信任都有代价；发现灵脉枯竭症的非自然痕迹，暗影楼第一次正面交锋，主角被迫暴露部分灵瞳能力；卷尾盟友背叛或牺牲，林玄意识到信任是有成本的。第三卷"阴谋与裂变"，情绪曲线是陡升陡降——真相碎片拼出全貌，灵管局高层与暗影楼的交易证据浮出，一名核心盟友的立场发生不可逆变化；主角修为突破至凝元境，但灵瞳过度使用导致第一次寿元透支；卷尾阴谋全貌拼出，林玄从"求生"转向"复仇"。第四卷"曝光与突围"，情绪曲线是高压释放——将高层交易证据公之于灵管局内部，引发权力震荡；主角突破筑基境，灵脉枯竭症暂时缓解；暗影楼楼主首次正面现身，与主角达成一次未分胜负的对峙；卷尾筑基境突破完成，但代价是灵瞳出现永久性损伤。第五卷"终局与抉择"，情绪曲线是悲壮收束——灵脉核心真相完全揭露，主角做出斩断/继承/摧毁的选择，付出不可逆代价；妹妹的命运得到最终安置，暗影楼残党被清算或蛰伏，灵管局进入新秩序；卷尾灵脉核心抉择完成，城市灵脉网络被重塑，林玄失去灵瞳但活下来了。

### Body
## 当前任务
林玄在地铁站出站口，主动第二次开启灵瞳，验证其看穿灵气流动的功能，并确认妹妹电话威胁与城市灵脉节点存在关联。

## 读者此刻在等什么
1) 读者期待林玄的"灵瞳"到底是什么能力，能不能真的帮他解决困境
2) 本章通过一次具体使用来展现灵瞳功能，同时让读者意识到这个金手指有代价（不能无脑用）

## 该兑现的 / 暂不掀的
- 该兑现：灵瞳功能 → 在地铁站柱子上看到绿色漩涡和地底抽吸现象，证实第1章所见非幻觉
- 该兑现：灵瞳代价 → 使用后出现轻微眩晕、元气消耗感，给读者"这是有限资源"的实感
- 暂不掀：暗影楼具体成员 → 只通过灵瞳看到异常能量波动，不直接出现人脸，留到第3章

## 日常/过渡承担什么任务
- 出站口排队人群 → 通过拥挤、嘈杂对比主角内心的紧张，强化"隐秘修真"的都市感
- 地铁站柱子特写 → 用视觉化描写（绿色漩涡、纹理）替代纯设定说明，让世界观落地
- 林玄独自行走 → 内心独白梳理妹妹电话与自身状况，为后续行动做决策铺垫

## 关键抉择过三连问
- 主角本章最关键的一次选择：
  - 为什么这么做？ → 妹妹电话被威胁，他必须确认灵瞳能否帮自己发现危险源头
  - 符合当前利益吗？ → 灵瞳是唯一的破局工具，不用等于等死
  - 符合他的人设吗？ → 沉稳内敛的人不会鲁莽，但会权衡后精准使用有限资源
- 对手/配角本章最关键的一次选择：
  - 为什么这么做？ → 无直接对手出场，但灵管局巡逻人员的存在感作为隐性阻力
  - 符合当前利益吗？ → 灵管局监控范围大，主角必须在他们眼皮底下行动
  - 符合他的人设吗？ → 巡逻人员按章办事，不主动惹事但保持警惕，符合"结构性对手"定位

## 章尾必须发生的改变
- 信息改变：确认地铁站柱子是灵脉节点，且正在被异常抽取能量
- 关系改变：林玄对妹妹的担忧从"被动接收信息"转为"主动调查"
- 物理改变：灵瞳使用1次，剩余2次/日，身体出现轻微不适感
