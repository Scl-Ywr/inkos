# 陈默 — 次要角色档案

## 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 陈默 |
| 年龄 | 24岁 |
| 身份 | 自由情报贩子，表面是网吧老板 |
| 修炼境界 | 开脉境后期（未筑基） |
| 能力 | 信息收集、人脉网络、基础法术 |
| 核心作用 | 林玄的信息源和盟友，提供关键情报 |

## 性格特征

- **圆滑世故**：在灰色地带生存多年，懂得如何与人打交道
- **重利但也重情**：做生意讲究互利，但对认可的人愿意帮忙
- **谨慎保守**：知道的事情多，但不会轻易透露
- **旁观者心态**：不轻易站队，但关键时刻会选边

## 背景故事

### 出身
- 曾是某个小修真家族的成员，家族覆灭后流落城市
- 靠收集信息和倒卖情报为生，建立了自己的情报网络
- 在老城区开了一家网吧，作为掩护和据点

### 与林玄的交集
- 最初是偶然相遇，林玄需要情报，陈默需要资源
- 逐渐形成互利合作关系
- 对林玄的灵瞳能力有所耳闻，但不知道详情
- 在关键时刻可能提供重要信息（如暗影楼据点、灵管局内部消息等）

## 人物弧线

| 阶段 | 状态 | 关键转变 |
|------|------|----------|
| 初期 | 情报贩子，互利合作 | 被林玄的坚韧打动 |
| 中期 | 有限信任，提供关键信息 | 可能卷入危险，被迫站队 |
| 后期 | 明确盟友关系 | 可能牺牲自己保护林玄或林雨 |
| 终局 | 结局取决于选择 | 可能存活，也可能成为牺牲品 |

## 人际关系

| 角色 | 关系 | 态度 |
|------|------|------|
| 林玄 | 客户→盟友 | 互利→有限信任→认可 |
| 苏晚晴 | 潜在信息源 | 保持距离，必要时交易 |
| 暗影楼 | 敌人 | 明确对立，但不正面冲突 |
| 灵管局 | 中立 | 不主动招惹，也不合作 |

## 说话风格

- 口语化，带点市井气
- 喜欢用"兄弟""哥们"称呼熟人
- 谈生意时精明，谈感情时真诚
- 情绪波动时会抽烟

## 关键场景标记

- 首次相遇：林玄需要调查暗影楼，找陈默买情报
- 提供关键信息：如灵管局内部消息、暗影楼据点位置
- 危机时刻：可能被迫卷入战斗，或为保护林玄而牺牲

## 悬念设计

- **家族覆灭的真相**：是否与暗影楼或灵管局有关？
- **终局选择**：他会选择帮助林玄到底，还是明哲保身？
- **情报网络的极限**：他能提供多少关键信息？有没有他不知道的秘密？
