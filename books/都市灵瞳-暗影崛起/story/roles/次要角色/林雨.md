# 林雨 — 次要角色档案

## 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 林雨 |
| 年龄 | 16岁 |
| 身份 | 高中生，林玄的妹妹 |
| 修炼境界 | 未觉醒（但可能有特殊体质） |
| 能力 | 未知（可能被暗影楼盯上是因为特殊灵根） |
| 核心作用 | 林玄的情感软肋，暗影楼的关键棋子（H002钩子） |

## 性格特征

- **乖巧懂事**：父母去世后与哥哥相依为命，早熟
- **依赖哥哥**：对林玄有强烈的依赖感，但也想证明自己
- **内心敏感**：察觉到哥哥的秘密，但不敢问
- **坚韧**：表面柔弱，内心有不服输的一面

## 背景故事

### 家庭变故
- 五年前父母"意外"身亡，与哥哥相依为命
- 哥哥承担了所有责任，她尽量不添麻烦
- 一直觉得哥哥最近有些奇怪，但说不出来

### 被盯上的原因
- 灵瞳首次觉醒时，林玄看到妹妹身上有追踪印记（H002）
- 暗影楼认为她可能是"纯灵体"或拥有特殊灵根
- 她的存在是林玄的软肋，也是暗影楼要利用的棋子

## 人物弧线

| 阶段 | 状态 | 关键转变 |
|------|------|----------|
| 初期 | 普通高中生，依赖哥哥 | 发现自己被神秘组织盯上 |
| 中期 | 开始察觉哥哥的秘密 | 被迫卷入修真界纷争 |
| 后期 | 从被保护者转向主动 | 可能觉醒特殊能力，成为关键变量 |
| 终局 | 站在哥哥身边 | 见证哥哥的最终选择 |

## 人际关系

| 角色 | 关系 | 态度 |
|------|------|------|
| 林玄 | 哥哥 | 绝对依赖，情感核心 |
| 苏晚晴 | 哥哥的朋友 | 警惕但逐渐信任 |
| 暗影楼 | 敌人 | 恐惧但逐渐反抗 |

## 说话风格

- 对哥哥温柔，偶尔撒娇
- 对陌生人保持距离
- 情绪波动时会沉默

## 关键场景标记

- 灵瞳首次看到追踪印记：地铁站，林玄第一次意识到妹妹的危险
- 被暗影楼盯上：可能遭遇跟踪或试探
- 觉醒时刻：可能在危机中被迫觉醒特殊能力

## 悬念设计

- **特殊体质**：她是否真的是"纯灵体"？还是暗影楼的误判？
- **觉醒潜力**：如果觉醒，她的能力是什么？
- **终局选择**：她会成为哥哥的助力，还是被敌人利用？
